firstName = name
Arturo = valu of the field "firstName"

&= Separates fields w/in the Query

===========

GET = Query is appended at the end of the URL
	= Search queries
	= Filtering options queries

LIMITATION
	= Cap for the no. of characters in a URL
	: Adding an "essay" or "feedback" string
		= Using GET places that whole essay into the URL stream. Which is No-no
	= Security issues
		= DO NOT!!!!! Put usernames and Passwords as
		Get!!!!

============

POST = Information / Query is stored within the request body

		+Content to be added to databases

		+Security and sensitive information
